### Manejo de variables
variable1 = 3
variable2 = variable1

variable1 = "pera"

variable2 = ????? --->Como todo lo que se almacena en variables son objetos, variable2 sigue apuntando al objeto al que apuntaba incialmente variable1 (2). Variable1 pasó a apuntar a otro objeto (pera) pero variable 2 sigue apuntando al objeto original. En otros lenguajes como java o C#, variable2 apuntaría al a la direccion de memoria de variable1, es decir que su valor sería "pera (no sería posible igual, ya que C# y Java es tipado,  y no se podría modificar el tipo de la variable)

### Programacion orientada a objetos vs programacion funcional
Programacion funcional mas performante, objetos mejor organizacion, aastraccion.

En Programacion funcional, el código se realiza de arriba hacia abajo, de izquierda a derecha
